# 엑셀 분석 요약

## 시트 역할

| 시트명 | 역할 분류 | 설명 |
| --- | --- | --- |
| 성적입력 | 입력용 | 시도, 행정구, 학교명, 생기부 관련 값, 내신, 모의고사 입력 셀을 모아 둔 시트 |
| 계산기 | 출력용 | 대학/전형/모집단위 원본 데이터와 티어, 점수, 컷, 판정 결과를 보여주는 최종 결과 시트 |
| 종합배치표 | 기준표용 | 학생부종합 전형의 상세계열별 A~E 티어 컷 테이블 |
| 종합보정값 | 보정값용 | 학생부종합 배치표를 생성하는 기준 fit, 티어 가감점, gap, offset 규칙 |
| 교과배치표 | 기준표용 | 학생부교과 전형의 상세계열별 A~E 티어 컷 테이블 |
| 교과보정값 | 보정값용 | 학생부교과 배치표를 생성하는 기준 fit, 티어 가감점, gap, offset 규칙 |
| 목록 | 드롭다운용 | 시도/행정구/학교명을 이름정의 범위로 관리하는 종속 드롭다운 원본 시트 |
| 고등학교DB | 학교 기준 데이터 | 학교별 인문/자연 총점, 티어, 유형 정보가 들어 있는 기준 테이블 |

## 입력 구조 분석

- `성적입력!C5`는 이름정의 `시도목록`을 사용합니다.
- `성적입력!C6`는 `INDIRECT("R_"&C5)`로 시도별 행정구 목록을 바꿉니다.
- `성적입력!C7`은 `INDIRECT("S_"&C5&"_"&C6)`로 학교 목록을 바꿉니다.
- `성적입력!C8`, `C9`는 `고등학교DB`에서 학교명을 `MATCH`로 찾고 `INDEX`로 인문/자연 티어를 읽어 옵니다.
- `성적입력!C13:C20`은 내신 지표, `F13:F21`은 모의고사 지표 입력 셀입니다.

## 계산기 시트 관찰

- `계산기!M`은 지원 행의 계열(`I`)에 따라 인문 또는 자연 학교 티어를 선택합니다.
- `계산기!N`은 점수 참조 셀입니다. 현재 검증한 버전에서는 실데이터 행 5,630건이 모두 `성적입력!C15`를 참조합니다.
- `계산기!R:S:T`는 상세계열(`J`)과 학교 티어(`M`)를 기준으로 종합배치표 또는 교과배치표에서 컷을 조회합니다.
- `계산기!P`는 입력 점수와 상향/적정/안정 컷을 비교해 `상향 / 소신 / 적정 / 안정`을 반환합니다.

## 웹 데이터 모델

- `regionTree`: `목록` 시트의 이름정의 기반 드롭다운을 웹용 계층 구조로 변환
- `schools`: `고등학교DB`를 학교 기준 JSON으로 정규화
- `admissions`: `계산기` 시트의 결과 행 메타데이터
- `comprehensiveAdjustments`, `schoolRecordAdjustments`: 보정값 시트의 규칙 블록
- `comprehensivePlacementTable`, `schoolRecordPlacementTable`: 원본 배치표 시트 값
- `validationSamples`: 기본 입력값 기준 대표 검증 케이스

## 확인한 수식 패턴

- `IF`, `IFERROR`
- `INDEX`, `MATCH`
- `SUBSTITUTE`
- 시트 간 직접 참조
- 티어 라벨 치환 후 기준값 조회
- 기준 fit + 티어 가감 + gap + offset 조합
